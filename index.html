About

Hemingway is a tool to easily manage, run, and collaborate on miscellaneous tasks.

Installation

Gemfile

$ gem 'hw'
$ bundle

Standalone

  
$ gem install hw

Usage

Retrieving community packages

The first thing that you'll want to do when installing Hemingway is to update all packages. When run for the first time, it will retrieve all available community packages.

  
$ hw update

Adding a custom package source

To add a custom package source you must provide a name and either the URL of the git repository or the local folder path.

  
$ hw add_source  
$ hw add_source private git@github.com:carrot/hw-packages-private.git
$ hw add_source personal ~/hw-personal-packages

With this approach, you have the ability to override existing packages. Note: When overriding packages, the most recently added sources hold greater priority.

After adding a source, you must update all package sources.

Updating sources

To update all sources simply run:

  
$ hw update
