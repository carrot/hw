# HW (Hemingway)

Package manager for miscellaneous tasks.

## Installation

Add this line to your application's Gemfile:

    gem 'hw'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hw

## Usage

    $ hw system update
